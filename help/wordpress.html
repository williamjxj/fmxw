CentOS 6.2: Install mongoDB server and PHP mongo extension

mongoDB: A good reference is here. The following are steps: 1) $ sudo vi /etc/yum.repos.d/10gen.repo: 
[10gen]
name=10gen Repository
baseurl=http://downloads-distro.mongodb.org/repo/redhat/os/x86_64
gpgcheck=0
enabled=1
2) $ yum install mongo-10gen mongo-10gen-server 3) /etc/mongod.conf, /etcrc.d/init.d/mongod $ sudo service mongod start/restart/stop $ sudo chkconfig mongod on 4) This MongoDB instance will store its data files in the /var/lib/mongo and its log files in /var/log/mongo, and run using the mongod user account. 5) $ sudo pecl install mongo $ sudo vi /etc/php.ini: 
[mongo]
extension=mongo.so
$ sudo service httpd restart 6) vi test_mongo.php: 

<php
$m = new Mongo();
print_r($m);
and return:
