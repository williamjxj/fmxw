$config = array(
 'header' => array(...)
);
function foo() {
  if (isset($config) && is_array($config['header'])) {
    //do some stuff.
  }
}
foo();
the foo() will not be executed because $config is unseen in the function. To correct it, add: global $config; to declare it, like this: 
function foo() {
  global $config;
  if (isset($config) && is_array($config['header'])) {
    //do some stuff.
  }
}
foo();
Right now it works.
